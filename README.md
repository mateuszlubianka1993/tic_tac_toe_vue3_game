# <img src="https://vuejs.org/images/logo.png" width="40" height="40" /> Vue 3 Tic Tac Toe

A modern, reactive Tic Tac Toe game built with **Vue 3**, **TypeScript**, and **Vite**.

## 🚀 Live Demo

Check out the live version here: **[[LINK](https://mateuszlubianka1993.github.io/tic_tac_toe_vue3_game/)]**

---

## ✨ Features

- **Global Game State:** Shared reactivity across components using the Singleton pattern in Composables.
- **Match History:** History tracking with persistent storage in `localStorage`.
- **Detailed Statistics:** Real-time calculated win rates and match logs.
- **Responsive UI:** Fully responsive design built with Tailwind CSS.
- **E2E Testing:** Critical user paths covered by Cypress.

---

## 🛠️ Tech Stack

- **Framework:** [Vue.js 3](https://vuejs.org/) (Composition API)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Routing:** [Vue Router](https://router.vuejs.org/)
- **Testing:** [Cypress](https://www.cypress.io/) (E2E) & [Vitest](https://vitest.dev/) (Unit)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 📦 Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/mateuszlubianka1993/vue-tic-tac-toe.git](https://github.com/mateuszlubianka1993/vue-tic-tac-toe.git)
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```

## 🧪 Testing

**Unit Tests:**

```bash
npm run test:unit
```

**E2E Tests:**

```bash
npm run test:e2e
```
